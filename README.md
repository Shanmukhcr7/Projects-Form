# Final Project Submission System

A modern Spring Boot SaaS dashboard for submitting and reviewing final projects.

## Features
* Glassmorphism & SaaS-style responsive forms
* Secure Admin Login Dashboard
* Integrated FontAwesome and Bootstrap 5
* MySQL Database with Spring Data JPA

## Tech Stack
* Java 17 + Spring Boot 3.2.x
* Thymeleaf + HTML5 + CSS3
* Maven
* MySQL

## Run
``nmvn spring-boot:run
``n
